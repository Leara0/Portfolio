Snake | Real-time, grid-based game

A classic Snake game implementation built entirely in the C# console, demonstrating real-time input handling, collision detection, and dynamic game state management. The game features smooth snake movement, progressive difficulty scaling, and responsive controls for an engaging retro gaming experience.

Technical Highlights

The application uses multithreading to handle simultaneous game rendering and user input. A background thread continuously listens for arrow key presses while the main thread manages the game loop, updating the snake's position, checking collisions, and rendering frames at calculated intervals.

One technical challenge I solved was creating responsive yet controlled input handling. The game needed to register direction changes immediately without allowing multiple inputs during a single move, which could cause the snake to reverse into itself. I implemented a two-stage direction system with a NextDirection buffer that queues player input and applies it at the start of each game loop iteration, combined with an IsOpposite validation method to reject illegal reversals.

Features

  • Multithreaded Input Handling: Background thread for continuous key listening while the main thread runs the game loop without blocking
  • Collision Detection: Boundary checking to detect when the snake hits walls and self-collision detection to end the game when the snake intersects its own body
  • Dynamic Difficulty: Speed modulation that increases based on score, with different sleep intervals for horizontal versus vertical movement to account for console character aspect ratios
  • Polished UI: Unicode box-drawing characters for clean borders, centered text formatting, alternating colored snake segments, and adaptive screen size validation

The game maintains state through a static Game class that stores snake coordinates as a list of tuples, current direction using an enum, score, and apple position. The snake grows by adding segments to the coordinate list when consuming apples, and movement is achieved by adding a new head position while removing the tail (unless growing).

Technologies: C#, Console API, Multithreading, Tuples, Enums, Unicode
